Introduction

Soul Anti-Cheat is a server-sided anti-cheat. It can protect your game from common exploits such as:

  • Speed Exploits
  • Teleport Exploits
  • Flying Exploits
  • Noclip Exploits
  • Root removal Exploits
  • Fling Exploits
  • SeatWeld Exploits

but you can create your own checks to counter other exploits!

Soul Anti-Cheat also has a Discord webhook feature!
