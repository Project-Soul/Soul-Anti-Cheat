Checks

A check is a module detecting anomalies in player behavior.
To be registered as a check, a check must have a '.check' extension.

Example:#

Fly.check

Requirements#

table Check.Connectionsfunction Check:Start(player: Player)

Make sure to add RBXScriptConnections in Check.Connections to avoid performance issues.

Example
Connections.OnHeartbeat = RunService.Heartbeat:Connect(function()
end)
